# GPU HBM Memory Contention Simulation

This example simulates the High Bandwidth Memory (HBM) resource contention on a GPU, a critical bottleneck for large AI models like vision encoders and language decoders. It demonstrates how different model components consume conceptual HBM capacity and how processing larger datasets can lead to increased processing times, analogous to bandwidth pressure.

## Language

`python`

## How to Run

Save the code as `main.py`. Run from your terminal: `python main.py`.
